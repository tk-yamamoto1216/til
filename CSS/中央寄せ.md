 # text-align :center
 
 ```
 <div class="outer">
  <div class="inner">
    Lorem ipsum dolor sit amet, consectetur adipiscing el
  </div>
</div>
```
```
.inner{
  text-align: center;
}
```

中身のテキストや画像といった**インライン要素**の中央寄せ  

インライン要素とは  
・横幅と高さの指定が可能  
・横幅の初期値が内容で決まる  
・横に並んでいく  
・marginは左右のみ、paddingは上下左右できるけど、上下にはくせがある  
・widthとheightは指定できない  

# margin: 0 auto;

 
 ```
 <div class="outer">
  <div class="inner">
    Lorem ipsum dolor sit amet, consectetur adipiscing el
  </div>
</div>
```
```
.inner  {
  width: 80%;
  margin: 0 auto;
  max-width: 500px;
}
```
・**widthの指定が必須**   
・単位「%」で指定すればレスポンシブWebデザインへの対応が可   
・PCビューで最大幅を決めるため、合わせてmax-widthを使うこともある
